The role's only hard metric. Tagged `general` so it survives both the data-eng and
delivery projections — capability build-out reads as adoption evidence to an
engineering audience and as program outcome to a delivery one.

than a closed result. Refresh the count as the program progresses.

cohort size only invites questions about the people who did not certify, which is
not this résumé's business and not something to record about identifiable
colleagues in a public repo.
