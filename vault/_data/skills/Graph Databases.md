ground-truth store in [[Virtual Knowledge Graph Connectivity]], the RDF retrievers
in [[Biomedical GraphRAG Bench]]), property graphs the exception —

Verizon. That is a decade-old, single-context exposure, not current fluency; M7
brings Neo4j back as hands-on work. Read `working` as covering RDF properly and
property graphs thinly.
